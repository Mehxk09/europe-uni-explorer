# Europe University Explorer

A calm, ad-free tool to browse European universities by country — filter by tuition, living costs, and English programmes, then shortlist and compare options.

## Features

- Browse **30+ European countries** with tuition and living cost badges
- **Search & filter** universities (English programmes, budget, country)
- **University detail pages** with links to official sites and programme portals
- **Shortlist** with notes (saved in your browser)
- **Compare** up to 4 universities side-by-side
- **Help me choose** wizard — 5 questions → suggested countries

## Prerequisites

- Node.js 20+
- npm

## Setup

```bash
cd euro-uni-explorer
npm install
npm run db:sync    # Fetches universities from HEI API (~2-5 min)
npm run dev        # Open http://localhost:3000
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run db:sync` | Sync countries & universities from [HEI API](https://hei.api.uni-foundation.eu/about) |
| `npm run build` | Production build |
| `npm run start` | Run production server |

## Data sources

- **Universities:** [HEI API](https://hei.api.uni-foundation.eu/) (European Higher Education Institutions)
- **Tuition, living costs, English notes:** Curated in `data/curated-countries.json` — always verify on official sites before applying

## Project structure

```
src/
  app/           # Next.js pages
  components/    # UI components
  db/            # Drizzle schema & SQLite connection
  lib/           # Queries, types, wizard logic
data/
  curated-countries.json
  euro-uni.db    # Generated by db:sync
scripts/
  sync-hei.ts    # HEI API sync script
```

## Note

Tuition and living cost figures are **estimates** for research purposes. Application deadlines, fees, and programme languages vary by university — always check official sources.
